# fe-bootstrap-architecture

## 定位

这是“前端项目初始化与基础架构” skill，负责新项目启动、已有项目结构审查、框架选型、基础目录设计和环境配置基线。

## 主要职责

- 新项目初始化
- 已有前端项目结构审查
- 框架选择，例如 React、Next.js、Vite、Vue
- 目录结构、模块边界、共享层设计
- 环境配置基线设计
- lint、format、类型检查、测试基线等工程基础设施规划

## 适合场景

- 从零开始搭建前端项目
- 现有项目前期架构梳理或重构
- 需要决定框架、构建方式或项目结构
- 需要统一环境配置和基础工程规范

## 不适合场景

- 纯页面样式调整
- 单个接口接入
- 仅做组件视觉优化

## 推荐使用方式

- 新项目时先用它确定技术和结构基线
- 老项目时先读取本地仓库结构，再决定是否需要调整
- 如果后续涉及设计实现、数据接入、测试与发布，再组合其他子 skill
