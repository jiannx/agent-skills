# fe-design-implementation

## 定位

这是“前端设计稿实现与 UI 落地” skill，负责把设计稿、截图或现有产品界面转成高一致度、可维护的前端实现。

## 主要职责

- 根据 Figma、Sketch、截图或现有产品实现 UI
- 响应式和移动端适配
- 基于 theme、token、语义变量、组件变体的样式落地
- 交互一致性控制
- 设计比对、截图校验和验收说明

## 适合场景

- 设计稿转代码
- 页面视觉实现
- 样式统一与设计修正
- 设计还原和截图比对

## 不适合场景

- 只做框架选型或架构初始化
- 只做数据层重构
- 只做发布或监控建设

## 推荐使用方式

- 优先通过 token、theme、shared components、variants 实现样式统一
- 如果存在设计稿，完成后需要做截图比对和差异说明
- 如果需要组件抽象或 feature 结构改造，可组合 `fe-component-feature-system`
